# Contexto de sesión — Piper Neo

Este paquete contiene el contexto útil para continuar en una conversación limpia sobre Piper Neo.

Se omitió intencionalmente la parte reciente del supuesto problema de lectura rara en textos largos, porque el usuario confirmó que el problema era ese texto específico y no quiere arrastrar ese hilo.

## Objetivo general del proyecto

Piper Neo es una evolución/fork de Piper TTS con:

- Server/API local.
- Soporte para formato `.neo`.
- App Electron/React para escritorio.
- Cliente visual para estudio, selección de voces/modelos, historial, chat LLM + TTS.
- Build/packaging para Windows y Docker.
- Enfoque en uso local/offline, clean code, diseño moderno y documentación.

## Preferencias del usuario

- Idioma: español.
- Prefiere entregables en ZIP sin `.git`.
- Hace commits manualmente desde GitHub Desktop.
- Quiere summaries/descriptions para commits en español.
- Prefiere clean code, modularidad, documentación y cambios bien explicados.
- No quiere strings tipo “ChatGPT” en commits o metadata.
- Valora diseño moderno, responsive, dark mode/offline, sin CDN cuando sea posible.
